typo3-crop-fal
==============

##What does it do?
It adds a link to awizard to each file reference, in the wizard you can crop your image.

##Requirements
 - ImageMagick
 - TYPO3 6.1.x

##Installation
Install it as extension in typo3! No configuration needed.
